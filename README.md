# enrichminion_qa_assignment

This repo contains QA deliverables for the EnrichMinion assignment (Signup, Login, Enrichment, Verification).

## Structure
- ui_test_cases.xlsx  - UI test cases (Signup, Login, Enrichment, Verification)
- api_test_cases.xlsx - API test cases (major endpoints + negative scenarios)
- bug_report.md       - Sample bug reports with RCA
- test_data.csv       - Provided Test_File.csv (first 100 rows included in UI/API test files)

## How to use
1. Download the files from this folder.
2. Open `ui_test_cases.xlsx` and `api_test_cases.xlsx` in Excel or Google Sheets to view test cases.
3. Use `test_data.csv` for bulk upload/enrichment tests.
4. Run Playwright/Postman automation samples if included (not included in this deliverable by default).
